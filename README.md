# dotfiles

Personal macOS dotfiles — fully automated setup for a productive development environment.

## What's included

| Component | Details |
|-----------|---------|
| Shell | zsh + Oh My Zsh + plugins |
| Prompt | Starship |
| Terminal | Ghostty |
| Editor | Zed |
| Package manager | Homebrew |
| Node.js | Volta |
| Containers | OrbStack |
| CLI tools | ripgrep, fd, bat, eza, fzf, tmux, and more |
| macOS settings | 50+ `defaults write` preferences |

## Prerequisites

- macOS (no Linux support)
- Internet connection
- Admin privileges (for Xcode CLT and Homebrew)

## Quick start

```bash
bash run.sh
```

This runs all scripts in order: bootstrap → install → symlink → macos, then `brew cleanup`.

## Scripts

| Script | Responsibility |
|--------|---------------|
| `run.sh` | Orchestrator — runs all scripts in sequence |
| `bootstrap.sh` | Xcode CLT, Homebrew, `brew bundle` |
| `install.sh` | fzf keybindings, git globals, Volta, GPG, Claude Code, Oh My Zsh + `.zshrc` |
| `symlink.sh` | Symlinks `config/*` files to their system paths |
| `macos.sh` | Applies macOS system preferences via `defaults write` |

Run individual scripts:

```bash
bash bootstrap.sh   # Xcode CLT + Homebrew + brew bundle
bash install.sh     # Tool config + Oh My Zsh + .zshrc
bash symlink.sh     # Symlink config files
bash macos.sh       # Apply macOS preferences
```

## Config files

All configs live in `config/` and get symlinked to their system locations. Existing files are backed up to `.bak` before being overwritten.

| Source | Destination |
|--------|------------|
| `config/zed/settings.json` | `~/.config/zed/settings.json` |
| `config/zed/keymap.json` | `~/.config/zed/keymap.json` |
| `config/ghostty/config` | `~/Library/Application Support/com.mitchellh.ghostty/config` |
| `config/git/config` | `~/.config/git/config` |
| `config/claude/settings.json` | `~/.claude/settings.json` |
| `config/starship.toml` | `~/.config/starship.toml` |
| `config/tmux/tmux.conf` | `~/.config/tmux/tmux.conf` |

## Shell setup

- **Shell:** zsh with Oh My Zsh
- **Config:** `~/.zshrc` is generated by `install.sh`
- **Plugins:** git, fzf, zsh-autosuggestions, zsh-syntax-highlighting
- **Prompt:** Starship (configured via `config/starship.toml`)

## macOS settings

`macos.sh` applies preferences across:

- **Dock** — auto-hide, icon size, hot corners
- **Finder** — show extensions, path bar, status bar, hidden files
- **Keyboard** — fast key repeat, disable autocorrect
- **Trackpad** — tap to click, tracking speed
- **Screenshots** — save location, disable shadows
- **Security** — require password immediately after sleep

## Manual steps

After running `bash run.sh`:

1. **Restart your session** — reload shell environment
2. **Enable FileVault** — System Settings → Privacy & Security → FileVault
3. **Authenticate Claude** — run `claude` and follow the login flow
4. **Link 1Password** — open 1Password and connect your account

## Adding new configs

1. Place the config file in `config/` (matching the desired path structure)
2. Add a `link` call in `symlink.sh`:

```bash
link "config/your-tool/config" "$HOME/.config/your-tool/config"
```

3. Run `bash symlink.sh` to apply
